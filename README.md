# Kijana Simama - Youth Stand Up

Official website for Kijana Simama, an NGO empowering in-school adolescents in Tanzania through mental health support, sexual and reproductive health education, and financial capacity building.

## About

Founded in 2025 and based in Dar es Salaam, Tanzania, Kijana Simama is dedicated to transforming the lives of young people through holistic education and support programs.

## Website Features

- 🏠 Home - Introduction and hero section
- ℹ️ About Us - Our story, mission, and vision
- 🎯 Services - Our 9 holistic programs
- 📢 Campaigns/Projects - Current initiatives
- 📅 Events - Upcoming and past events
- 🖼️ Gallery - Photo showcase
- 💝 Donate/Support - Ways to contribute
- 📞 Contact - Get in touch
- ❓ FAQs - Common questions

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS v4
- Vite
- React Router DOM
- Framer Motion
- Recharts

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Maintenance Mode

A reusable site-down page is available at:

```text
/site-down
```

To temporarily show the maintenance page for the whole website, set this environment variable before building/deploying:

```text
VITE_SITE_DOWN=true
```

To bring the website back, remove the variable or set it to:

```text
VITE_SITE_DOWN=false
```
